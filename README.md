# team10-isa-mrs
ISA/MRS project

## Prerequisites
    a) Java
    b) Postgres SQL
    c) IntelliJ
    d) Node.js

## Setup
1. Clone this repository
2. Open the `/backend` folder in IntelliJ
3. Open the `/frontend` folder in terminal

### Running backend
From IntelliJ run AppApplication

### Running frontend
Run this command in the opened terminal:
```sh
npm run serve
```
